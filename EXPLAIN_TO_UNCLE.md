## üìã Glossary - What All These Terms Mean

### Construction Terms:

| Term | Full Form | What It Means |
|------|-----------|---------------|
| **RFI** | Request For Information | When site needs clarification from consultant/architect. "What detail should we follow here?" |
| **BOQ** | Bill of Quantities | List of all materials and quantities needed for the project |
| **PMC** | Project Management Consultant | Company hired to oversee the project |
| **QA/QC** | Quality Assurance / Quality Control | Checking that work meets standards |
| **STR** | Structural | Structural drawings (columns, beams, slabs) |
| **ARC** | Architectural | Architectural drawings (layout, finishes) |
| **MEP** | Mechanical, Electrical, Plumbing | All the services (AC, wiring, pipes) |
| **R1, R2, R3** | Revision 1, 2, 3 | Version numbers of drawings. R3 is newer than R2. |
| **Grid** | Grid Reference | The letter-number system to locate elements (A1, B2, C3) |
| **Rebar** | Reinforcement Bar | Steel bars inside concrete for strength |

### Tech Terms:

| Term | Full Form | What It Means |
|------|-----------|---------------|
| **AI** | Artificial Intelligence | Computer that can understand and respond like a human |
| **LLM** | Large Language Model | The technology behind ChatGPT - understands text |
| **API** | Application Programming Interface | How different software talks to each other |
| **WhatsApp API** | - | Official way to build automated WhatsApp services |
| **Embeddings** | - | Converting text into numbers so computer can search/compare |
| **PDF** | Portable Document Format | Standard file format for documents |
| **Dashboard** | - | Visual screen showing all important information |
| **SaaS** | Software as a Service | Software you pay monthly for (like Netflix) |
| **MRR** | Monthly Recurring Revenue | How much money comes in every month |

### Business Terms:

| Term | Full Form | What It Means |
|------|-----------|---------------|
| **ROI** | Return on Investment | How much you get back for what you spend |
| **MVP** | Minimum Viable Product | Simplest version of product that works |
| **Pilot** | Pilot Program | Trial period to test with real customers |
| **Churn** | - | When customers stop paying |
| **ARPU** | Average Revenue Per User | How much each customer pays on average |
